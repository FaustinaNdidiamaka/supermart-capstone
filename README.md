# SuperMart Data Analytics Capstone Project

## 📊 Project Overview

SuperMart is a Nigerian retail chain operating across six regions (North, South, East, West, North-Central, South-South), serving customers in 30 cities. This project involves querying the sales database to generate insights on revenue performance, employee effectiveness, customer behaviour, and inventory.

### Dataset Summary

| Metric | Value |
|--------|-------|
| Customers | 800 |
| Employees | 35 |
| Products | 68 |
| Orders | 2,000 |
| Order Items | 5,508 |
| Time Period | January 2021 – June 2024 |
| Currency | Nigerian Naira (₦) |
